3.
 
**
添加重要日期
**
：点击格言模块的 "+" 按钮，输入标题和日期。
4.
 
**
管理备忘录
**
：点击备忘录模块的 "+" 按钮添加，双击列表项编辑/查看。
5.
 
**
数据库位置
**
：数据存储在 
`
%APPDATA%/MyOrg/sd-desktop-assistant/data.db
`
。
